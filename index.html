swap
root
▼
Create a 2G swap file, persist across reboots, set swappiness=10.
$ curl -sL runsh.de/swap.sh | bash
custom size: SWAP_SIZE=4G bash -c "$(curl -sL runsh.de/swap.sh)"
what it does
  • Creates /swapfile via fallocate (or dd as fallback)
  • chmod 600, mkswap, swapon
  • Appends to /etc/fstab for persistence
  • Sets vm.swappiness=10 in /etc/sysctl.conf

bbr
root
▼
Enable BBR TCP congestion control. Improves throughput on high-latency links. Kernel ≥ 4.9.
$ curl -sL runsh.de/bbr.sh | bash
what it does
  • Checks kernel version ≥ 4.9
  • modprobe tcp_bbr
  • Sets net.core.default_qdisc=fq
  • Sets net.ipv4.tcp_congestion_control=bbr
  • Persists via /etc/sysctl.conf

ssh-harden
root
▼
Disable root login & password auth, change SSH port to 22222.
$ curl -sL runsh.de/ssh-harden.sh | bash
what it does
  • Verifies SSH key exists before proceeding
  • Backs up /etc/ssh/sshd_config
  • Sets Port=22222, PermitRootLogin=no, PasswordAuthentication=no
  • Sets MaxAuthTries=3, X11Forwarding=no
  • Updates UFW rules if installed
⚠ Ensure SSH key auth works before running. Test in a new terminal after!
