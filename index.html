runsh.de
Shell scripts, ready to run
// basics
⚡
init
New server bootstrap — mirrors, common tools, timezone (Asia/Shanghai), UFW setup.
$ curl -sL runsh.de/init.sh | bash
🔄
update
Update all packages, remove unused deps, clean apt cache, vacuum logs.
$ curl -sL runsh.de/update.sh | bash
💾
swap
Create a 2G swap file and persist it across reboots. Set swappiness=10.
$ curl -sL runsh.de/swap.sh | bash
custom size: SWAP_SIZE=4G ... | bash
🚀
bbr
Enable BBR TCP congestion control for better network throughput. Kernel ≥ 4.9.
$ curl -sL runsh.de/bbr.sh | bash
📊
info
Display system overview — OS, CPU, memory, disk, network, Docker status.
$ curl -sL runsh.de/info.sh | bash
⏱
bench
Quick benchmark — CPU speed, disk read/write, network download test.
$ curl -sL runsh.de/bench.sh | bash
// security
🔒
ssh-harden
Disable root login & password auth, change port to 22222. Requires key auth first!
$ curl -sL runsh.de/ssh-harden.sh | bash
🛡
ufw
Configure UFW firewall: deny all incoming, allow SSH / HTTP / HTTPS.
$ curl -sL runsh.de/ufw.sh | bash
custom port: SSH_PORT=22222 ... | bash
// dev environment
🐳
docker
Install Docker Engine + Compose plugin. Adds current user to docker group.
$ curl -sL runsh.de/docker.sh | bash
🐚
zsh
Install zsh + oh-my-zsh with autosuggestions and syntax highlighting.
$ curl -sL runsh.de/zsh.sh | bash
⬢
node
Install Node.js via nvm. Defaults to LTS v22. Easily switch versions later.
$ curl -sL runsh.de/node.sh | bash
custom version: NODE_VERSION=20 ... | bash
🌐
nginx
Install Nginx with gzip, tuned worker count, and a clean default page.
$ curl -sL runsh.de/nginx.sh | bash
🔐
cert
Get a free SSL cert via acme.sh (Let's Encrypt). Auto-installs Nginx config.
$ DOMAIN=example.com EMAIL=you@x.com curl -sL runsh.de/cert.sh | bash
